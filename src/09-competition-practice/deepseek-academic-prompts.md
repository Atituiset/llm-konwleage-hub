# 50 个 DeepSeek 学术指令

> **资料来源**：《50个顶级的DeepSeek学术论文指令》
> **适合人群**：需要提升学术效率的研究者
> **难度**：⭐⭐（容易）

---

## 1. 文献综述类

### 1.1 领域综述
```
请帮我总结 [领域] 近 5 年的研究进展。要求：
1. 列出 10 篇关键论文（含作者、年份、核心贡献）
2. 用表格对比各方法的优缺点
3. 分析该领域的发展趋势
4. 指出当前的研究空白和未来方向
```

### 1.2 论文对比
```
请对比以下两篇论文的异同：
论文 A：[标题]
论文 B：[标题]
要求从方法、实验、结论三个维度分析
```

### 1.3 相关工作整理
```
请帮我整理以下主题的相关工作：
主题：[主题]
要求按方法类别分类，每类列出 3-5 篇代表性论文
```

### 1.4 研究趋势分析
```
分析 [领域] 近 3 年的研究热点变化。
要求：
1. 用时间线展示技术演进
2. 每个阶段说明代表性工作和驱动力
3. 预测未来 2 年的发展方向
```

## 2. 论文写作类

### 2.1 摘要撰写
```
请根据以下内容撰写学术论文摘要（200-250 词）：
[研究背景]
[方法概述]
[主要结果]
[核心贡献]
```

### 2.2 引言撰写
```
请为以下研究主题撰写引言部分：
主题：[主题]
要求：
1. 介绍研究背景和意义
2. 综述现有方法的不足（引用 5-8 篇文献）
3. 明确本文的创新点
4. 概述论文结构
```

### 2.3 方法描述
```
请用学术论文风格描述以下方法：
[方法描述]
要求：
1. 使用准确的数学符号
2. 分步骤说明
3. 与现有方法对比说明创新点
```

### 2.4 实验结果描述
```
请根据以下实验数据撰写结果分析段落：
[实验数据]
要求：
1. 突出关键发现
2. 与基线对比
3. 分析可能的原因
4. 指出局限性
```

## 使用建议

1. **根据具体需求修改模板**：不要直接复制，根据你的研究领域调整
2. **提供足够的上下文**：模型需要了解背景才能给出高质量输出
3. **对生成内容保持批判性思维**：AI 可能产生幻觉，关键事实必须人工核查
4. **最终内容需人工审核**：AI 辅助提高效率，但不能替代人的判断
5. **分步骤使用**：复杂任务拆分为多个简单指令，逐步完善
